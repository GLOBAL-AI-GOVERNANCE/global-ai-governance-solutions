# Executive Dashboard Metrics

## Core Metrics

- Total AI systems inventoried
- Percent with named owners
- Percent risk-classified
- Percent with evidence complete
- Percent with monitoring active
- Percent with shutdown path
- Number of High/Critical/Frontier systems
- Number of vendor AI systems
- Number of open governance gaps
- Number of incidents
- Number of systems paused or shut down

## Health Thresholds

Suggested thresholds:

- Green: 90%+ completion
- Yellow: 70% to 89%
- Red: Below 70%

## Executive Interpretation

The goal is not a perfect dashboard.

The goal is decision visibility.

Executives need to know where risk is concentrated, where ownership is missing, and where deployment should pause until evidence catches up.
