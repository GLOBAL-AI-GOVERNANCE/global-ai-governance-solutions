# Vendor AI Review Scorecard

Use this scorecard before purchasing, deploying, renewing, or expanding vendor AI.

| Area | Score 0-4 | Evidence Required |
|---|---:|---|
| Data protection |  | Data handling, retention, training-use terms |
| Security controls |  | SOC 2, ISO, pen test, architecture, logging |
| Privacy controls |  | DPIA, privacy policy, deletion rights |
| Auditability |  | Logs, exportable records, review process |
| Model update transparency |  | Release notes, change notice terms |
| Incident notification |  | Contractual notification window |
| Subprocessor transparency |  | Subprocessor list and update process |
| Termination rights |  | Data deletion/return terms |
| Liability/accountability |  | Indemnity, warranty, limitation terms |
| Shutdown support |  | Disable process, fallback plan |

## Decision Bands

- 34–40: Strong, proceed with standard controls
- 25–33: Proceed with restrictions and legal review
- 15–24: High concern; require remediation
- 0–14: Do not deploy without major correction
