# v1.2 Automation Layer

Global AI Governance Solutions v1.2 adds lightweight automation for inventory validation, risk tiering, policy-as-code checks, and governance reporting.

## Purpose

v1.0 established the Governance OS.  
v1.1 added the Practical Toolkit.  
v1.2 adds the Automation Layer.

## What This Layer Does

- Validates AI system inventory files
- Calculates preliminary risk tiers
- Checks required governance evidence
- Flags missing owners, monitoring, and shutdown paths
- Produces governance gap reports
- Provides policy-as-code rules for repeatable review
- Adds GitHub workflow scaffolding for automated governance checks

## Operating Law

No AI system moves faster than ownership, evidence, authority, and control.

## Important Note

These tools support governance review. They do not replace human judgment, legal review, security review, privacy review, or executive accountability.
