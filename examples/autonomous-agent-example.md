# Example: Autonomous Agent

## System

Internal research agent with browser and document summarization tools.

## Risk Tier

Moderate

## Approved Use

- Search approved internal knowledge sources
- Summarize documents
- Draft research memos
- Flag source gaps

## Prohibited Use

- Sending external emails
- Making purchases
- Accessing sensitive data repositories
- Modifying production systems
- Operating without logs
- Expanding tool access without approval

## Required Controls

- Least privilege
- Time-bound sessions
- Tool allowlist
- Logging
- Human review before external use
- Monitoring
- Shutdown path

## Approval Decision

Approved with restricted permissions.
