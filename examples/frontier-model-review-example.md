# Example: Frontier Model Review

## System

Advanced model with autonomous tool use and access to sensitive enterprise workflows.

## Risk Tier

Frontier

## Why Frontier

The system has high capability, broad tool access, and potential to affect security, operations, data, and public trust.

## Required Review

- Board-level review
- Executive owner assignment
- Red team testing
- Cybersecurity review
- Privacy review
- Legal review
- Safety evaluation
- Misuse evaluation
- Model behavior evaluation
- Tool-use control review
- Shutdown test

## Minimum Controls

- Least privilege
- Human authorization for high-impact actions
- Tool allowlist
- Rate limits
- Session limits
- Sensitive data controls
- Full logging
- Real-time monitoring
- Incident response
- Verified shutdown path

## Approval Decision

Not approved for production until evidence demonstrates control, accountability, and safe operation.
